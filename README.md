# rust-color-visuals
Cool rust visuals for Shipwrecked project 4
Controls:
    Space: pause
    R: reseed noise
    [ / ]: scale down/up
    , / .: z_step down/up
    / / =: force down/up
    9 / 0: friction down/up
    F / G: fade up/down
    C: cycle color mode
    S: save frame as frame_000000.png
